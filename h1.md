# h1 Hei Ansiblen maailma

## SSH public key - Login without password
- SSH asennetaan päivittämällä järjestelmä ja lataamalla SSH
- Enabloidaan SSH, jotta se käynnistyy automaattisesti uudelleenkäynnistyksien yhteydessä
- Testataan localhost-yhteys, jolloin SSH yhdistetään **tähän tietokoneeseen**
- Luodaan avain SSH-keygen:illä, jolloin .ssh -hakemistoon ilmestyy tiedostot jossa avaimet ovat
- Kopioidaan ID kohdekoneelle (localhost) jolloin avain lisätään authoerized_keys -tiedostoon
- Lopputuloksena voidaan yhdistään kohteeseen (localhost) ilman salasanan syöttöä

Lähde: https://terokarvinen.com/ssh-public-key-login-without-password/


## Hello Ansible
- Asennetaan Ansible ja tehdään sille oma hakemisto
- Luodaan **hosts.ini** -tiedosto, johon lisätään käsiteltävät koneet (tässä vain localhost)
- Luodaan **ansible.cfg** -tiedosto, jossa voidaan määritellä automaattisia asetuksia komento-toistojen välttämiseksi
- Luodaan **site.yml** -tiedosto (playbook), jossa määritellään asetukset "mitä suoritetaan ja missä" (hostit ja niiden roolit)
- Luodaan **roles/hello/tasks/main.yml** -tiedosto, johon kirjoitetaan suoritettava tehtävä
- Lopputuloksena **ansible-playbook site.yml** -komento:
  - Lukee hosts.ini -> localhost
  - Suoritetaan site.yml roolimääritykset -> hello
  - Suorittaa main.yml:ssä määritellyn tehtävän -> luo tiedoston

Lähde: https://terokarvinen.com/hello-ansible/
